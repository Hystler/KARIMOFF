## Примечания

- Данные взяты из браузерно загруженного состояния страницы 'vike_pageContext', где сайт хранит категории и товары.
- Для товаров с несколькими модификаторами в seed выбран самый дешевый базовый вариант; остальные варианты сохранены в JSON/CSV и перечислены выше.
- 'old_price', 'product_url' и 'image_url_original' сохранены в JSON/CSV для контроля, но не пишутся в 'products', потому что текущая таблица проекта их не содержит.
- Seed не удаляет и не отключает старые товары автоматически. В конце SQL есть закомментированный optional cleanup для старых placeholder-позиций.
